# 05 — 培訓計畫

> **對口窗口:** 培訓負責人
> **最晚交付日:** Phase 5 W3
> **用途:** 建立培訓追蹤系統，讓資深經營者能看到新人的培訓進度

---

## 為什麼這份很重要

如果培訓內容沒有結構化，系統只能追蹤「完成/未完成」，但追蹤的是什麼？不知道。
這就像考試沒有考卷一樣。

---

## 你需要準備什麼

### 1. MEGA 培訓課程大綱（0.5.1）

**檔案名稱建議:** `mega_training_outline.md`

請提供完整的培訓課程結構：

| 週次 | 課程主題 | 學習目標 | 時長 | 形式 |
| :--- | :--- | :--- | :--- | :--- |
| W1 | 品牌認識與產品知識 | 能說出 3 款核心產品的特色 | 2 小時 | 線上/實體 |
| W1 | 個人品牌建立 | 能發出第一篇社群貼文 | 1 小時 | 實作 |
| W2 | 私訊互動技巧 | 能用引導式對話回覆 3 位朋友 | 2 小時 | 角色扮演 |
| W2 | 問卷使用教學 | 能邀請 1 人填寫問卷並解讀結果 | 1 小時 | 實作 |
| W3 | 商談技巧入門 | 能完成 1 次模擬商談 | 2 小時 | 角色扮演 |
| W3 | 異議處理基礎 | 能回答 5 個最常見異議 | 1 小時 | 練習 |
| W4 | 客戶管理 | 能使用系統追蹤 3 位客戶 | 1 小時 | 實作 |
| W4 | 成果整理與回顧 | 完成第一個月總結報告 | 1 小時 | 文件 |
| （...） | | | | |

#### 需要回答的問題

- 培訓總共幾週？
- 有沒有進階培訓？（完成 MEGA 之後還有什麼？）
- 培訓期間有沒有指導上線？一對一還是小組？
- 是否有線上課程影片？影片連結可以提供嗎？

### 2. 每週任務清單（0.5.2）

**檔案名稱建議:** `weekly_tasks.xlsx`

每週具體要完成的任務：

| 週次 | 任務 | 完成標準 | 驗證方式 | 必做/選做 |
| :--- | :--- | :--- | :--- | :--- |
| W1 | 發第 1 篇社群貼文 | 貼文已發布 | 截圖上傳 | 必做 |
| W1 | 列出 30 位潛在名單 | 名單已建立 | 系統內確認 | 必做 |
| W1 | 觀看產品知識影片 | 影片看完 | 系統記錄 | 必做 |
| W2 | 私訊 5 位朋友 | 已發出訊息 | 截圖上傳 | 必做 |
| W2 | 邀請 2 人填問卷 | 問卷已提交 | 系統內確認 | 必做 |
| W2 | 完成問卷解讀練習 | 正確率 ≥ 70% | 線上測驗 | 必做 |
| （...） | | | | |

#### 重要提醒

- 每個任務要有「完成標準」，不是只有做/沒做
- 驗證方式要可執行（截圖？系統記錄？上線確認？）
- 區分「必做」和「選做」，新人不要一次壓太多

---

### 3. 評量標準（0.5.3）

**檔案名稱建議:** `evaluation_criteria.md`

請定義：

#### A. 什麼算「完成培訓」？

| 條件 | 門檻 |
| :--- | :--- |
| 必做任務完成率 | ≥ 80% |
| 模擬商談通過 | 至少 1 次 |
| 實際商談參與 | 至少 1 次（含旁聽） |
| 問卷解讀正確率 | ≥ 70% |
| （其他...） | |

#### B. 什麼算「需要額外輔導」？

| 信號 | 建議動作 |
| :--- | :--- |
| 連續 2 週未完成任務 | 上線一對一訪談 |
| 模擬商談連續 2 次未通過 | 加強角色扮演練習 |
| 問卷解讀正確率 < 50% | 重看產品知識影片 |
| （其他...） | |

#### C. 什麼算「放棄」？

- 連續多少天沒登入？
- 連續多少週沒完成任務？
- 要不要有正式的「退出」機制？

## 交付方式

1. 將檔案放入此資料夾（`05_training_program/`）
2. 通知產品經理
3. PM 會安排審查，確認任務可以被系統追蹤

## 驗收標準

- [ ] 培訓大綱涵蓋至少 4 週
- [ ] 每週至少 3 個具體任務
- [ ] 每個任務有明確的完成標準
- [ ] 「完成培訓」的門檻已定義
- [ ] 「需要輔導」的信號已定義
