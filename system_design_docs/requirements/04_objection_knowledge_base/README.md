# 04 — 異議知識庫

> **對口窗口:** 資深經營者（有豐富商談經驗的人）
> **最晚交付日:** Phase 2 W3
> **用途:** 建立 AI 商談助手的知識庫，讓新手經營者在遇到客戶異議時能快速找到合適的回覆

---

## 為什麼這份很重要

新手經營者最怕的就是商談時被問倒。
你腦中有 100 種回答方式，但新人只有 3 種。
這份知識庫就是把你的經驗變成新人可以直接用的武器。

---

## 你需要準備什麼

### 1. 異議分類清單（0.4.1）

**檔案名稱建議:** `objection_categories.xlsx`

請依以下分類列出你遇過的所有客戶異議：

| 分類 | 異議原文 | 出現頻率 | 嚴重度 |
| :--- | :--- | :--- | :--- |
| **價格** | 「太貴了」 | 非常高 | 中 |
| 價格 | 「我在別家買比較便宜」 | 高 | 中 |
| 價格 | 「我現在手頭比較緊」 | 高 | 高 |
| **信任** | 「這跟直銷有什麼不一樣」 | 非常高 | 高 |
| 信任 | 「你有專業背景嗎」 | 中 | 中 |
| 信任 | 「網路上有人說不好」 | 低 | 高 |
| **時機** | 「我再想想」 | 非常高 | 低 |
| 時機 | 「現在不是好時機」 | 高 | 低 |
| 時機 | 「等我忙完這陣子再說」 | 高 | 低 |
| **家人** | 「我要跟老公/老婆商量」 | 高 | 中 |
| 家人 | 「我家人不相信這個」 | 中 | 高 |
| **產品** | 「我之前吃保健品都沒用」 | 高 | 中 |
| 產品 | 「味道我不確定能接受」 | 中 | 低 |
| 產品 | 「成分安全嗎」 | 中 | 中 |
| （其他...） | | | |

#### 欄位說明

- **異議原文**：客戶實際會說的話，越口語越好
- **出現頻率**：非常高 / 高 / 中 / 低
- **嚴重度**：高 = 如果處理不好就掰了 / 中 = 可以挽回 / 低 = 只是拖延

> **最低要求：30 條異議**。越多越好，100 條以上最理想。

---

### 2. 每條異議的建議回覆（0.4.2）

**檔案名稱建議:** `objection_responses.xlsx`

每條異議請提供至少 3 種語氣的回覆：

| 異議 | 溫和版回覆 | 專業版回覆 | 直接版回覆 |
| :--- | :--- | :--- | :--- |
| 「太貴了」 | 「我完全理解。其實很多朋友一開始也覺得投資不小，但他們後來發現...」 | 「如果換算成每日的健康投資，大約是一杯手搖飲的價格。而且跟去醫院的費用比...」 | 「坦白說，好的東西確實不便宜。但你過去在減重上花的錢加起來，可能比這個還多...」 |
| 「我再想想」 | 「當然，這是很重要的決定。如果你有任何問題，隨時可以問我...」 | 「好的。你最想多了解哪個部分呢？是產品效果、費用，還是其他？」 | 「理解。不過我想確認一下，你是對哪個部分比較猶豫？這樣我可以先幫你整理資訊...」 |
| ... | ... | ... | ... |

#### 重要提醒

- **請用你自己平常講話的方式寫**，不要用官方行銷語言
- **不需要完美**，先寫出來，我們會一起修
- 如果某條異議你自己也覺得很難回，就寫「這題我也覺得難，通常會請上線處理」

---

### 3. 成功與失敗案例（0.4.3）

**檔案名稱建議:** `objection_cases.md`

請寫出 5-10 個你印象深刻的商談案例：

#### 成功案例範本

```
案例編號：S-001
客戶背景：45 歲女性，職業護理師，有 15 年減重經驗
主要異議：「我吃過太多保健品了，都沒有用」
你怎麼處理的：
  1. 先同理：「你的經驗我完全能理解...」
  2. 提出差異：「我們的產品跟一般保健品不同的是...」
  3. 提供證據：「這是某位護理師朋友使用後的變化...」
結果：當場決定先試用一個月
成功關鍵：同行見證最有說服力
```

#### 失敗案例範本

```
案例編號：F-001
客戶背景：38 歲男性，工程師，太太介紹來的
主要異議：「我不相信直銷」
你怎麼處理的：
  1. 嘗試解釋公司模式
  2. 提供產品研究資料
結果：對方禮貌拒絕，之後不再回覆
失敗原因：一開始就進入辯論模式，沒有先處理情緒
如果重來會怎麼做：先不提直銷，聚焦在健康改善的結果
```

> **失敗案例比成功案例更有價值**。因為 AI 要學會避開什麼坑。

## 交付方式

1. 將檔案放入此資料夾（`04_objection_knowledge_base/`）
2. 通知產品經理
3. PM 會安排 2 小時工作坊：用角色扮演的方式，把更多隱性經驗挖出來

## 驗收標準

- [ ] 至少 30 條異議，涵蓋價格/信任/時機/家人/產品 5 大類
- [ ] 每條異議至少有 1 種回覆（理想是 3 種語氣）
- [ ] 至少 3 個成功案例 + 3 個失敗案例
- [ ] 回覆內容使用口語化表達，非行銷官方語言
