## 你需要準備什麼

### 1. 健康問卷欄位表（0.2.1）

**檔案名稱建議:** `health_questionnaire_fields.xlsx`

請用以下格式列出每一題：

| 題號 | 群組 | 題目 | 題型 | 選項 | 是否必填 | 隱私等級 | 跳題邏輯 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| Q1 | 基本資料 | 請問您的性別 | 單選 | 男/女/其他/不願透露 | 是 | 低 | — |
| Q2 | 基本資料 | 請問您的年齡區間 | 單選 | 20-29/30-39/40-49/50-59/60+ | 是 | 低 | — |
| Q3 | 健康現況 | 目前最困擾的健康問題 | 多選 | 體重管理/三高/消化/睡眠/精力/皮膚/其他 | 是 | 中 | — |
| Q4 | 健康現況 | 是否有慢性病或長期用藥 | 是否+文字 | 是/否 + 補充說明 | 否 | 高 | 選「是」才顯示補充 |
| ... | ... | ... | ... | ... | ... | ... | ... |

#### 欄位說明

- **群組**：基本資料 / 健康現況 / 健康經驗 / 生活習慣 / 意向
- **題型**：單選 / 多選 / 文字 / 數字 / 是否 / 是否+文字 / 評分量表
- **隱私等級**：
  - 低：一般資訊，所有人都要填
  - 中：敏感資訊，可選擇不填，但填了分析更準
  - 高：非常敏感，預設不顯示，消費者主動選擇才出現
- **跳題邏輯**：什麼條件下才顯示這題（留空 = 一定顯示）

> **目前 PRD 已有初版 14 題**（見 `02_synergy_ai_sow.md` §3.1）。
> 請以此為基礎，補充或修改。

### 2. FORM 背景問卷欄位表（0.2.2）

**檔案名稱建議:** `form_background_fields.xlsx`

FORM = Family / Occupation / Recreation / Money

| 類別 | 題號 | 題目 | 題型 | 選項 | 是否必填 | 備註 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| Family | F1 | 婚姻狀態 | 單選 | 已婚/未婚/其他 | 否 | |
| Family | F2 | 子女數 | 數字 | 0-10 | 否 | |
| Family | F3 | 家中是否有人在意健康管理 | 是否 | 是/否 | 否 | 用來判斷家庭支持度 |
| Occupation | O1 | 職業類別 | 單選 | 上班族/自營/自由業/家管/退休/學生/其他 | 否 | |
| ... | ... | ... | ... | ... | ... | ... |

#### 重要提醒

FORM 資料是經營者在商談「前」蒐集的，不是消費者自己填的。
所以這份表格要回答的問題是：

- 經營者需要知道哪些背景資訊？
- 這些資訊用來判斷什麼？（例如：知道收入區間是為了判斷承擔能力）
- 哪些欄位是「一定要問」、哪些是「問到最好」？

### 3. 問卷流程圖（0.2.3）

**檔案名稱建議:** `questionnaire_flow.md` 或繪圖檔

請描述：

```
消費者點開問卷連結
    │
    ├─→ 第 1 段：基本資料（3-4 題）
    │       └─→ 填完後顯示「已完成 25%」
    │
    ├─→ 第 2 段：健康現況（4-5 題）
    │       ├─→ 如果選了「慢性病」→ 顯示補充題
    │       └─→ 填完後顯示「已完成 50%」
    │
    ├─→ 第 3 段：健康經驗（3-4 題）
    │       └─→ 填完後顯示「已完成 75%」
    │
    └─→ 第 4 段：意向（3-4 題）
            └─→ 完成 → 顯示感謝頁 + 初步回饋
```

需要回答的問題：
- 分幾段？每段幾題？
- 中斷後可以續填嗎？（建議：可以）
- 填完後消費者看到什麼？（純感謝？還是初步建議？）
- 問卷連結的有效期？（永久？7 天？）

### 4. 隱私等級分類（0.2.4）

**檔案名稱建議:** `privacy_classification.md`

請確認以下設計是否符合你的需求：

| 隱私等級 | 行為 | 適用欄位舉例 |
| :--- | :--- | :--- |
| **低** | 預設顯示，必須填寫 | 性別、年齡區間、居住地、運動頻率 |
| **中** | 預設顯示，可選擇跳過 | BMI、預算區間、減重經驗 |
| **高** | 預設隱藏，點「我願意提供更多資訊」才顯示 | 慢性病、用藥紀錄 |

需要確認：
- 這三級分法夠不夠？需要更細嗎？
- 消費者跳過中/高隱私題目後，經營者會看到「未填寫」標示。這樣可以嗎？
- 有沒有法規面的顧慮？（例如：蒐集健康資訊需要額外同意書？）
