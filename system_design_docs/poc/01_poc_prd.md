## 1. 文件目的

本文件是完整 PRD 的 POC 縮減版，僅涵蓋 POC 階段交付的 3 個模組、12 個功能。完整產品尚有模組 3（商談支援）、模組 4（客戶管理）、模組 5（成果/轉介紹），POC 不實作。

---

## 2. POC 目標用戶

| 用戶類型 | 角色說明 | POC 涉及場景 |
| :--- | :--- | :--- |
| 消費者（外部用戶） | 潛在客戶，從認識品牌到填問卷接受評估 | 填問卷、收到評估報告 |
| 經營者（內部用戶） | 第一線業務人員 | 發文吸引、問卷判讀、名單分級 |
| 資深經營者（管理者） | 團隊主管 | 目標設定、績效看板、培訓追蹤 |

## 4. Module 1：AI 獲客行銷模組

### Feature 1.1：貼文生成引擎（P0）

| 欄位 | 內容 |
| :--- | :--- |
| User Story | As 經營者, I want to 輸入素材（照片+關鍵詞）後自動生成社群貼文文案和圖片, so that 我不用花時間想內容也能持續發文 |
| 主要用戶 | 經營者 |
| 輸入 | 素材照片、關鍵詞、目標平台（FB / IG / X / LINE） |
| 輸出 | AI 生成文案 + AI 生成/處理圖片 |
| 商業規則 | 符合品牌語調；各平台字數限制；可排程自動發布至少 2 個平台 |

**驗收標準**

| # | 驗收標準 |
| :--- | :--- |
| AC1 | 輸入素材後 < 30 秒產生文案 |
| AC2 | 文案品質經人工評估 ≥ 80% 可用率（50 篇樣本） |
| AC3 | 圖片可直接使用或僅需微調 |
| AC4 | 排程後成功發布到目標平台 |

---

### Feature 1.2：SEO 內容頁（P2）

| 欄位 | 內容 |
| :--- | :--- |
| User Story | As 經營者, I want to 輸入健康關鍵字後自動產出 SEO 優化長文, so that 潛在客戶搜尋時能找到我 |
| 主要用戶 | 經營者 |
| 輸入 | 健康管理相關關鍵字 |
| 輸出 | SEO 優化長文 + SERP 預覽 |
| 商業規則 | 包含 meta tags、結構化標題、關鍵字密度 2–3% |

**驗收標準**

| # | 驗收標準 |
| :--- | :--- |
| AC1 | 輸入關鍵字後 < 60 秒產出文章 |
| AC2 | 文章結構符合 SEO 基本規範 |
| AC3 | 可預覽 SERP 呈現 |

## 5. Module 2：AI 問卷評估模組

### Feature 2.1：健康問卷系統（P0）

| 欄位 | 內容 |
| :--- | :--- |
| User Story | As 消費者, I want to 在網頁上分段填寫健康問卷且可跳過隱私問題, so that 我不會覺得壓力太大而放棄 |
| 主要用戶 | 消費者 |
| 輸入 | 消費者填答 |
| 輸出 | 完成的問卷資料 |
| 商業規則 | 分段填答；進度保存；隱私保護（可跳過敏感題） |

**驗收標準**

| # | 驗收標準 |
| :--- | :--- |
| AC1 | 支援中斷續填 |

| AC3 | 填答完成率 ≥ 80%（測試場景） |

---

### Feature 2.2：問卷結果研判表（P0）

| 欄位 | 內容 |
| :--- | :--- |
| User Story | As 消費者/經營者, I want to 問卷完成後自動收到健康評估摘要, so that 雙方都知道初步狀況 |

| AC2 | 消費者版與教練版格式正確且有所區分 |
| AC3 | 摘要生成時間 < 15 秒 |

## 7. 非功能需求（POC 等級）

| 類別 | 規範 | 備註 |
| :--- | :--- | :--- |
| 效能 — 頁面載入 | < 3 秒 | 首次載入 |
| 效能 — API（非 LLM） | < 1 秒 | P95 |
| 效能 — API（LLM 呼叫） | < 15 秒 | 含 retry |
| 瀏覽器支援 | Chrome + Safari | 最新兩個主要版本 |
| 裝置支援 | 響應式 Web（手機 / 平板 / 桌機） | 不做原生 App |
| 安全性 | API token 不硬編碼；基本 input validation；問卷隱私模式 | 非 production 等級 |
| 資料規模 | ≤ 100 經營者、≤ 1,000 消費者 | POC 規模 |

## 8. POC 限制與取捨

| 限制 | 說明 |
| :--- | :--- |
| 單一組織 | 不支援多租戶，所有用戶屬於同一組織 |
| 僅 Web | 不做 App（iOS / Android） |
| 無離線模式 | 所有功能須連線使用 |
| LLM 幻覺風險 | AI 生成內容需人工複核，UI 應顯示免責提示 |
| MEGA 整合備案 | MEGA 培訓 API 不可用時以 CSV 匯入替代 |
| 不含客戶管理 | 模組 4（客戶管理）POC 不實作 |
| 不含商談支援 | 模組 3（商談支援）POC 不實作 |
| 不含成果/轉介紹 | 模組 5（成果/轉介紹）POC 不實作 |
