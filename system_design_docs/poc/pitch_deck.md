## Slide 12: 為什麼值得這筆投資

### ROI 試算框架

**人力節省（直接成本）**

| 情境 | 節省估算 |
|---|---|
| 每位經營者每天節省 1–2 小時行政作業 | 若時薪 NT$500，一個月省 15,000–30,000 |
| 10 人團隊，人均節省 1 小時/天 | 一個月省 150,000 工時成本 |

**轉化率提升（業績影響）**
- 結構化問卷 + AI 分級 → 精準跟進，不再廣撒漁網
- 即使轉化率只提升 5%，業績影響以倍數計算

**複製速度（組織資產）**
- 新人透過系統 30 分鐘上手，而非數週摸索
- 成功模式數據化，可複製到下一個市場

**數據資產（長期價值）**
- 問卷數據、行為數據是公司的長期資產
- POC 累積的數據，進入正式版立即可用

## Appendix B: 技術架構

### 已驗證技術棧

```
前端 (React 18/19 + Vite)
  ├─ 模組 1：shadcn/ui
  └─ 模組 2/6：Tailwind v4 + Apple Design Tokens

後端 (FastAPI + Python 3.12)
  ├─ uv 套件管理
  ├─ PostgreSQL 資料庫
  └─ LiteLLM (統一 LLM 介面)
       └─ Gemini 2.0 Flash（主要模型）

自動化 (n8n)
  ├─ 多平台內容發布流程
  └─ 問卷後續自動觸發

基礎設施
  ├─ Docker Compose（本地開發）
  └─ 雲端部署（TBD，Google Cloud / Railway）
```

**為何選這個技術棧？**
- FastAPI：Python 生態最快的 API 框架，原生 async
- LiteLLM：未來可切換不同 LLM 供應商，不被綁架
- n8n：視覺化自動化流程，業務人員可自行修改規則
- PostgreSQL：成熟穩定，支援 JSON 欄位儲存彈性資料
