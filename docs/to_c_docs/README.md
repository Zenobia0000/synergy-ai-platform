## TL;DR（懶得看完先看這裡）

### 我們要做什麼

**Synergy AI — 教練成交副駕駛**：一套給 Synergy WorldWide 教練使用的 AI 工具，涵蓋「從陌生人填問卷、到商談前 AI 摘要、到成交後自動跟進」的完整閉環。

### 四個 MVP 功能

1. **智能健康問卷**（手機填 5 分鐘、AI 自動分析）
2. **AI 商談摘要**★（教練商談前 5 分鐘打開就能看完的單頁簡報）
3. **陽春版 CRM**（客戶管理）
4. **自動跟進提醒**（LINE 推播為主、Email 備援）

### 時程

- **8 週** MVP 開發
- **4 週** Pilot（3 位教練實測）
- **共 12 週** 後交付驗收報告

### 投資

- Pilot 期月運營成本 **< 2,500 NTD**
- 工程開發費用另依合約

### 預期回收

- 教練成交率 **15% → 20%**（+33%）
- 商談準備時間 **30 分鐘 → 5 分鐘**（-83%）
- 48 小時跟進率 **< 40% → ≥ 80%**（+100%）
