---

## 一、一句話說明

**讓一位新教練，也能像 10 年老手一樣準備商談、準時跟進、成交客戶。**

做法是：從「陌生人填問卷」到「成交後跟進」，把整個漏斗**自動化 + AI 化**，教練只要專心做最人性化的部分——商談與成交。

---

## 二、我們想解決什麼問題

### 現場實況

| 現況 | 問題 |
|:---|:---|
| 問卷用 Google 表單 / App Script | 資料散落、無法自動分析、每份要手動看 |
| 商談前靠教練記憶 + 手抄筆記 | 新教練不知道該問什麼、老教練記不清每位客戶 |
| 跟進靠教練自己設手機提醒 | 80% 的客戶在商談後 48 小時內被遺忘 |
| 客戶名單散在 LINE + Excel | 看不到整體漏斗、找不到該跟的人 |

### 業績影響

- 中段教練（佔 80%）成交率約 **15%**，遠低於頂尖教練
- 客戶首購後 3 個月流失率 > **40%**
- 新教練 onboarding 時間長、培訓成果不穩定

---

## 三、我們要打造的解方

### 核心口訣

> **「商談前 5 分鐘打開手機，就能準備好這場商談。」**

### 第一階段 MVP 範圍（4 個功能閉環）

```
1️⃣ 智能健康問卷  →  2️⃣ AI 商談摘要  →  3️⃣ 客戶管理  →  4️⃣ 自動跟進
```

| # | 功能 | 解決的問題 |
|:-:|:---|:---|
| 1 | **智能健康問卷** | 客戶 5 分鐘填完 → 系統自動生出健康摘要 + 教練用名單 |
| 2 | **AI 商談摘要** ★ | 教練商談前打開就能看：痛點、推薦產品、預期異議、切入點 |
| 3 | **陽春版 CRM** | 一張可搜尋可篩選的客戶清單，狀態一目了然 |
| 4 | **自動跟進提醒** | 商談後 48 小時 / 7 天 / 30 天，LINE 自動提醒教練 |

★ = 最高價值功能

## 四、預期帶來的商業價值

### 可量化指標（Pilot 8 週後驗證）

| 指標 | 目前 | 目標 | 提升幅度 |
|:---|:---:|:---:|:---:|
| 中段教練商談成交率 | 15% | **≥ 20%** | +33% |
| 教練商談準備時間 | 30+ 分鐘 | **≤ 5 分鐘** | -83% |
| 48 小時跟進執行率 | < 40% | **≥ 80%** | +100% |
| 問卷完成率 | — | **≥ 50%** | 新基線 |
| 問卷轉商談率 | — | **≥ 30%** | 新基線 |

### 對組織的長期價值

1. **經驗可複製**：資深教練的商談 Playbook 變成系統提示，新人快速上手
2. **資料資產化**：未來半年累積客戶資料庫，支援成果轉介與潛力經營者識別
3. **擴張基礎**：架構預留跨品牌能力，未來可擴張至 USANA、Herbalife 等
4. **合規強化**：結構化商談紀錄，降低日後法規風險

## 五、第一期「明確不做」的範圍

避免範圍蔓延、專注驗證核心，以下**明確延後**至 Phase 2 或之後：

| 不做項目 | 為什麼延後 |
|:---|:---|
| AI 貼文生成、SEO 內容 | 生成品質難評估、對 MVP 成交沒立即貢獻 |
| 私訊機器人 / 自動回覆 | 需要量變才有意義 |
| 見證故事自動生成 | 需要幾十個成功案例才有料 |
| 轉介紹話術 / 潛力經營者識別 | 等 MVP 跑 6 個月累積資料 |
| 團隊績效儀表板、月目標管理 | 只服務 MEGA 層，工程成本高 |
| 培訓追蹤系統 | Phase 5 再做 |
| 執行追蹤、衛教推播 | 客戶成交後才用得到 |
| 多語系、跨品牌客製 | Phase 3 |

## 六、時程與投資

### 開發時程（8 週）

```
Week 0  Kick-off ┬─ Pilot 教練確認
                 ├─ LINE OA 申請送件 ★
                 └─ 開發環境建置
Week 1-2 問卷系統       ─────────▶
Week 3-4 AI 商談摘要    ─────────▶
Week 5   CRM + 提醒排程 ───▶
Week 6   3 位 Pilot 教練內測
Week 7-8 迭代與驗收
        ─────────────────────────▶
               Phase 2 Gate（Go/No-Go）
```

★ LINE Official Account 審核需 1-2 週，必須在 Week 0 Day 1 送件

### 預算範圍

| 項目 | 月度成本（Pilot 期） |
|:---|:---|
| 雲端基礎設施（Supabase、Railway、Vercel） | ~1,000 NTD |
| AI 服務（Gemini API） | ~200-300 NTD |
| LINE Messaging API | ~800 NTD |
| Email 備援（Resend） | 免費 |
| 網域 + 雜項 | ~60 NTD |
| **Pilot 期月運營總計** | **< 2,300 NTD** |

工程開發成本另計，依合約。

## 七、試點計畫（Pilot）

### 試點對象

**Synergy WorldWide 3 位教練**：
- 1 位資深 MEGA
- 1 位中階經營者
- 1 位新人（入行 < 1 年）

### 試點方式

- 連續使用 **4-6 週**
- 每週回報實際使用心得
- Week 2 深度訪談（改進重點）
- Week 8 驗收五大指標

### Phase 2 啟動條件

五大指標中至少 **3 項達標** + 至少 1 位 Pilot 教練**願意付費續用**，才進入 Phase 2 擴張（Lead 獲客層、多品牌架構）。

## 八、風險與應對

| 風險 | 影響 | 應對方式 |
|:---|:---|:---|
| LINE OA 審核延遲 | 阻塞 Week 5 提醒功能 | Day 1 送件；未過審先走 Email |
| 教練採用度低 | KPI 無法驗證 | Week 0 教學、Week 2 訪談調整 |
| AI 摘要品質不夠 | 成交率提升不明顯 | Week 2 品質評估，必要時升級更強 AI |
| 個資合規疑慮 | 法規風險 | 問卷聲明「非醫療診斷」+ 客戶主動刪除權 |

## 九、成功後的下一步（Phase 2 藍圖）

MVP 驗收達標後，接續擴充：

1. **Lead 獲客層**（M1 基礎）：問卷連結主動分享系統、多平台 Lead 收集
2. **執行追蹤層**（M4 進階）：客戶重生計畫、衛教推播、續購提醒
3. **成果轉介層**（M5）：見證自動整理、潛力經營者識別
4. **團隊管理層**（M6）：月目標、績效儀表板、培訓追蹤
5. **跨品牌擴張**：USANA 亞太 → Herbalife 拉美 → Amway
