## 1. 消費者旅程地圖 (Customer Journey Map)

### 1.1 旅程總覽

```
看見改變 → 搜尋比較 → 私訊接觸 → 健康評估 → 篩選分流 → 商談 → 試用/成交 → 健康管理 → 成果呈現 → 轉介紹
   │           │          │          │          │         │        │           │           │          │
Awareness  Interest   Consideration                   Conversion          Retention              Advocacy
```

### 1.2 詳細旅程

#### Stage 1：看見改變（Awareness）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 在 FB/IG/LINE 看到朋友分享健康改善成果 |
| **行為** | 瀏覽貼文、點讚、留言、截圖 |
| **想法** | 「他怎麼變這麼多？」「是真的嗎？」 |
| **情緒** | 好奇、半信半疑 |
| **接觸點** | 社群貼文、限動、見證照 |
| **AI 介入** | AI 生成貼文文案、B/A 對照圖自動排版 |
| **系統動作** | — |

#### Stage 2：搜尋與比較（Interest）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 主動搜尋品牌、產品、經營者資訊 |
| **行為** | Google 搜尋、瀏覽粉專、看留言評價 |
| **想法** | 「這跟其他保健品有什麼不同？」「有沒有負評？」 |
| **情緒** | 謹慎、比較心態 |
| **接觸點** | SEO 內容頁、粉專、Google 搜尋結果 |
| **AI 介入** | FAQ 機器人自動回答常見問題、SEO 內容生成 |
| **系統動作** | — |

#### Stage 3：私訊接觸（Interest → Consideration）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 決定私訊詢問、或被限額活動吸引 |
| **行為** | 傳私訊、留 Email/LINE、報名活動 |
| **想法** | 「問問看應該不會怎樣吧」「免費評估感覺不錯」 |
| **情緒** | 試探、有點期待 |
| **接觸點** | Messenger/LINE/IG DM |
| **AI 介入** | 私訊接待助理（意圖辨識 + 引導留資） |
| **系統動作** | 建立 Lead（status: new → contacted），分派給經營者 |

#### Stage 4：健康評估（Consideration）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 受邀填寫健康問卷 |
| **行為** | 填答問卷（可選擇隱私模式、分段填答） |
| **想法** | 「這些問題我不太想全部回答」「填完會怎樣？」 |
| **情緒** | 有戒心但願意嘗試 |
| **接觸點** | 問卷頁面（Web） |
| **AI 介入** | 隱私模式提示、分段引導、填答後即時回饋 |
| **系統動作** | 建立 QuestionnaireResponse，觸發 AI 分析，Lead status → questionnaire_done |

#### Stage 5：篩選與分流（Consideration）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 系統自動分析問卷結果 |
| **行為** | （消費者不感知此階段） |
| **想法** | — |
| **情緒** | — |
| **接觸點** | — |
| **AI 介入** | 問卷分析、名單分級、分流規則匹配 |
| **系統動作** | 更新 Lead priority，執行分流動作（預約/培養/轉上線/試用品） |

#### Stage 6：商談（Conversion）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 與經營者 + 上線進行 2:1 商談 |
| **行為** | 見面或視訊、聽方案介紹、提出疑慮 |
| **想法** | 「適不適合我？」「會不會太貴？」「可以信任嗎？」 |
| **情緒** | 期待但謹慎、可能有壓力感 |
| **接觸點** | 面談/視訊/LINE 通話 |
| **AI 介入** | 經營者端：商談摘要、異議處理建議 |
| **系統動作** | Lead status → consultation_booked，記錄商談結果 |

#### Stage 7：試用與成交推進（Conversion）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 商談後仍有疑慮，提供試用品 |
| **行為** | 收到試用品、嘗試產品、回報感受 |
| **想法** | 「味道還行」「好像有感覺」「再想想」 |
| **情緒** | 體驗中、逐漸建立信任 |
| **接觸點** | 試用品寄送、回訪電話/訊息 |
| **AI 介入** | 試用回訪提醒、回覆建議 |
| **系統動作** | 記錄試用狀態，定時回訪提醒 |

#### Stage 8：執行健康管理計畫（Retention）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 成交後啟動重生計畫 |
| **行為** | 每日 checkin、執行飲食/運動/產品使用計畫 |
| **想法** | 「希望有效」「有人陪我比較不會放棄」 |
| **情緒** | 有動力但需要持續支持 |
| **接觸點** | Web 計畫頁面、LINE 提醒 |
| **AI 介入** | 每日提醒、衛教推播、進度鼓勵 |
| **系統動作** | Lead → Customer，建立 Plan，每日 checkin 記錄 |

#### Stage 9：成果呈現（Retention）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 計畫完成，產出成果 |
| **行為** | 拍照記錄、查看數據變化 |
| **想法** | 「真的有差耶！」「我想分享給朋友」 |
| **情緒** | 成就感、感恩 |
| **接觸點** | 成果報表頁面、B/A 對照 |
| **AI 介入** | 自動生成成果報表、B/A 圖文排版 |
| **系統動作** | Plan status → completed，生成成果數據 |

#### Stage 10：轉介紹與經營轉化（Advocacy）

| 項目 | 內容 |
| :--- | :--- |
| **情境** | 滿意度高，被邀請轉介紹 |
| **行為** | 分享成果給朋友、推薦經營者 |
| **想法** | 「這麼好的東西要分享」「我也想做這個事業」 |
| **情緒** | 熱情、認同 |
| **接觸點** | 社群分享、轉介紹表單 |
| **AI 介入** | 滿意度判斷、轉介紹邀請腳本、潛力經營者識別 |
| **系統動作** | 建立新 Lead（source: referral），評估經營者潛力 |

## 2. 經營者旅程地圖 (Operator Journey Map)

### 2.1 日常工作流

```
[每日啟動]
    │
    ├─→ 查看今日提醒 ──→ 回訪提醒 / 追蹤提醒 / 商談提醒
    │
    ├─→ 社群發文 ──→ AI 生成文案 → 選擇/修改 → 發布
    │
    ├─→ 回覆私訊 ──→ AI 推薦話術 → 引導填問卷
    │
    ├─→ 問卷判讀 ──→ AI 分析結果 → 決定跟進策略
    │
    ├─→ 商談準備 ──→ AI 生成客戶摘要 → 準備異議處理
    │
    ├─→ 客戶追蹤 ──→ 計畫執行追蹤 → 衛教推播
    │
    └─→ 查看績效 ──→ 今日/本週/本月進度
```

### 2.2 新手經營者首月旅程

| 週次 | 核心任務 | AI 支援 | 系統功能 |
| :--- | :--- | :--- | :--- |
| W1 | 建立個人品牌基礎 | 貼文文案模板、拍照腳本 | 社群發文工具 |
| W2 | 開始私訊互動 | 回覆話術推薦、對話引導 | 私訊管理 |
| W3 | 邀請填問卷、首次商談 | 問卷邀請話術、商談摘要 | 問卷系統、預約系統 |
| W4 | 追蹤跟進、嘗試成交 | 異議處理建議、跟進提醒 | 客戶管理、提醒系統 |

## 3. 資深經營者旅程地圖 (Leader Journey Map)

### 3.1 月度管理週期

```
[月初]
    │
    ├─→ 設定月目標 ──→ 個人目標 + 團隊目標
    │
    ├─→ 檢視上月績效 ──→ 漏斗分析 → 找出卡關點
    │
    └─→ 培訓規劃 ──→ 新人任務分派

[每週]
    │
    ├─→ 團隊績效檢視 ──→ 各成員進度 → 落後者提醒
    │
    ├─→ 培訓進度追蹤 ──→ 學習完成度 → 實作練習
    │
    └─→ 協助商談 ──→ 2:1 商談支援

[每日]
    │
    ├─→ 查看異常告警 ──→ 活躍度下降 / 成交率異常
    │
    └─→ 個人名單經營 ──→ 同經營者日常流程
```

### 3.2 關鍵決策點

| 情境 | 決策 | AI 支援 |
| :--- | :--- | :--- |
| 新人入職 | 分派培訓計畫 | 培訓模板推薦 |
| 成員拜訪率下降 | 介入輔導或調整策略 | 異常提醒 + 建議動作 |
| 團隊成交率低 | 分析卡在哪個環節 | 漏斗分析 + 環節比較 |
| 成員請求商談協助 | 判斷是否加入 2:1 | 客戶摘要 + 難度評估 |
| 月底績效落後 | 加速跟進或調整目標 | 進度預測 + 建議調整 |

## 5. 情緒曲線

### 消費者情緒曲線

```
高 ┤                                                    ★ 成果呈現
   │                                                   ╱
   │                                              ╱───╱
   │   ╲                                     ╱───╱
   │    ╲         ╱─╲                   ╱───╱
   │     ╲───╱──╱   ╲─────╱──╲────╱───╱
   │              私訊   問卷   商談  試用   執行計畫
低 ┤
   └─────────────────────────────────────────────────────→ 時間
     看見  搜尋  接觸  評估  分流  商談  成交  陪跑  成果  推薦

   低點：問卷填答（隱私疑慮）、商談（壓力）
   高點：看見改變（好奇）、成果呈現（成就感）
```

### 設計啟示

| 低點 | 設計對策 |
| :--- | :--- |
| 問卷填答 | 隱私模式、分段填答、即時回饋、明確告知用途 |
| 商談壓力 | 不強制見面、可選視訊、經營者接受異議處理訓練 |
| 計畫中期（倦怠） | 里程碑鼓勵、每日提醒、社群支持 |
