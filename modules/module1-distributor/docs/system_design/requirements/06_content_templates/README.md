# 06 — 內容模板素材

> **對口窗口:** 行銷窗口 + 資深經營者
> **最晚交付日:** Phase 4 W3
> **用途:** 作為 AI 內容生成引擎的訓練範本，決定 AI 生成內容的品質與風格

---

## 為什麼這份很重要

AI 生成內容的品質 = 你給它的範本品質。
如果你給它 10 篇好的貼文範例，它就能寫出類似品質的文案。
如果你什麼都不給，它只會寫出通用的、沒有溫度的內容。

---

## 你需要準備什麼

### 1. 社群貼文範例（0.6.1）

**資料夾建議:** `social_post_examples/`

請提供各平台各 10 篇「效果好」的貼文：

#### Facebook（10 篇）

每篇請提供：
- 貼文文字全文
- 配圖（如有）
- 互動數據（讚/留言/分享數）
- 發布日期
- 為什麼你覺得這篇效果好？

#### Instagram（10 篇）

同上，額外提供：
- Hashtag 策略
- 是 Feed / Reels / Story？

#### X / Twitter（10 篇）

同上，注意：
- 字數限制內的精華版本
- 有沒有用 Thread？

#### LINE OA（10 則）

- 推播內容全文
- 開信率/點擊率（如有）
- 是圖文訊息還是純文字？

> **沒有 40 篇也沒關係**。有多少給多少，但請標註哪些是你覺得「最好的」。

### 2. 私訊對話範例（0.6.2）

**資料夾建議:** `dm_conversation_examples/`

請提供至少 20 段真實對話（可匿名）：

#### 格式

```
對話編號：DM-001
平台：LINE
對方背景：35 歲女性，看到貼文主動私訊
結果：成功邀約填問卷

---

對方：你好～我看到你最近瘦好多，是怎麼做到的？
我：謝謝你注意到！其實我是做了一套完整的健康管理計畫...
對方：什麼樣的計畫啊？
我：簡單來說就是...（後續對話）
...
```

#### 需要的對話類型

| 類型 | 數量 | 說明 |
| :--- | :--- | :--- |
| 主動詢問 → 成功邀約 | 至少 5 段 | 對方主動來問，你成功引導到填問卷 |
| 主動詢問 → 未成功 | 至少 3 段 | 對方有興趣但最後沒繼續 |
| 你主動開場 → 成功 | 至少 3 段 | 你主動跟朋友提起 |
| 你主動開場 → 未成功 | 至少 3 段 | 主動出擊但碰壁 |
| 異議出現在私訊中 | 至少 3 段 | 對方在私訊中就開始質疑 |
| 久未聯繫的舊客回訪 | 至少 3 段 | 跟已購買但一陣子沒聯絡的人重新接觸 |

> **隱私處理**：請將對方真實姓名改為化名，移除可識別個資。

### 3. 見證故事範例（0.6.3）

**資料夾建議:** `testimony_examples/`

請提供至少 10 則已發布的見證內容：

每則包含：

**檔案名稱建議:** `education_content_inventory.xlsx`

請列出所有可用的衛教素材：

| 編號 | 標題 | 類型 | 主題分類 | 適合誰看 | 連結/位置 | 品質評價 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| EDU-001 | 為什麼你需要一氧化氮 | 文章 | 心血管 | 新客戶 | (連結) | 好 |

- 主題：心血管/體重管理/營養知識/運動/生活習慣/產品使用

### 5. 品牌語調指南（0.6.5）

**檔案名稱建議:** `brand_voice_guide.md`

請回答以下問題：

#### A. 品牌個性
- 用 3 個形容詞描述品牌的說話風格：____, ____, ____
- 如果品牌是一個人，他/她是什麼樣的人？

#### B. 語調規範

| 場景 | 語調 | 範例 |
| :--- | :--- | :--- |
| 社群貼文 | ？（輕鬆/專業/親切/...） | 「今天要跟大家分享...」 |
| 私訊回覆 | ？ | 「嗨～謝謝你的訊息...」 |
| 衛教內容 | ？ | 「根據研究顯示...」 |
| 商談場景 | ？ | 「我理解你的考量...」 |

#### C. 禁用詞/禁止做法

| 禁用 | 原因 |
| :--- | :--- |
| 例：「保證有效」 | 法規不允許療效宣稱 |
| 例：「不買會後悔」 | 壓力銷售會破壞信任 |
| 例：直接攻擊競品名稱 | 容易引發法律糾紛 |
| （其他...） | |

#### D. 鼓勵用的詞彙/句型

- 你們常用來鼓勵客戶的話是什麼？
- 有沒有品牌獨有的用語？
