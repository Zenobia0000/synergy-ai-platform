# 01 — 領域知識盤點

> **對口窗口:** 領域專家（熟悉 Synergy 產品與經營模式的人）
> **協作者:** 產品經理
> **最晚交付日:** Phase 1 啟動前 2 週
> **用途:** 讓開發團隊理解業務，設計正確的資料模型與流程

---

## 你需要準備什麼

### 1. 產品目錄與適用情境（0.1.1）

**檔案名稱建議:** `product_catalog.xlsx`

請提供所有可銷售產品的清單，包含：

| 欄位 | 說明 | 範例 |
| :--- | :--- | :--- |
| 產品名稱 | 正式名稱 | ProArgi-9+ |
| SKU / 編號 | 公司內部編號 | SYN-001 |
| 產品分類 | 心血管/體重管理/營養補充/... | 心血管保健 |
| 適用情境 | 什麼人適合用 | 三高族群、久坐上班族 |
| 價格帶 | 單品/組合價 | NT$3,500 / 月 |
| 組合包內容 | 如果有組合包，裡面有什麼 | ProArgi-9+ x2 + Mistify x1 |
| 使用方式簡述 | 一天吃幾次、怎麼吃 | 每日 2 包，溫水沖泡 |

> **至少需要**：完整的在售產品清單。如果有已下架但舊客還在用的產品也請列出。

---

### 2. 客戶分群定義（0.1.2）

**檔案名稱建議:** `customer_segments.md`

請回答以下問題：

- 你們目前怎麼分類客戶？（例如：新客/舊客/VIP？或依產品線分？）
- 有沒有「高價值客戶」的定義？怎麼判斷？
- 經營者通常怎麼稱呼不同階段的客戶？（例如：名單、準客戶、會員、夥伴）
- 有沒有「不適合的客戶」？什麼情況會建議對方不要購買？

> **不需要理想化的分群**。我要的是你們「現在實際在用」的分法，即使很粗糙也沒關係。

### 4. 現行作業流程紀錄（0.1.4）

**檔案名稱建議:** `current_workflow.md` 或螢幕錄影

這是最重要的一項。請描述（或錄影示範）：

#### A. 獲客流程
- 你現在怎麼發社群貼文？用什麼工具？
- 有人私訊之後，你怎麼處理？記在哪裡？
- 名單管理用什麼？（Excel？手機備忘錄？LINE 記事本？紙本？）

#### B. 評估與商談流程
- 問卷是用紙本還是 Google Form？還是口頭問？
- 商談前你怎麼準備？
- 2:1 商談的實際運作方式？

#### C. 成交後流程
- 客戶購買後，你怎麼追蹤他的使用狀況？
- 重生計畫怎麼執行？用什麼記錄？
- 多久回訪一次？用什麼提醒自己？

#### D. 團隊管理流程（如果你是資深經營者）
- 你怎麼知道團隊成員有沒有在做事？
- 你怎麼追蹤他們的進度？
- 新人培訓怎麼進行？

> **最好的方式**：請錄一段 30 分鐘的影片，邊操作邊講解。
> 沒時間錄影的話，用文字寫也可以，但請盡量具體。
> 「不好意思現在很亂」也沒關係——我們就是要改善這個亂。

## 常見問題

**Q: 我不確定某些產品還在不在售？**
A: 先列出來，標註「待確認」，比漏掉好。

**Q: 我們現在的流程很亂，沒有標準做法怎麼辦？**
A: 這正常。請描述你自己（或你認為做得最好的那個人）的做法就好。

**Q: 有些術語我自己也說不清楚定義？**
A: 寫下你理解的版本，訪談時我們一起釐清。
